# myslam
This project is based on _slambook14--gaoxiang_ chapter13.

## Quick Start
### 1. dependence
* opencv with opencv_contrib (version 3.4)
* DBoW3
* Sophus
* g2o
* glog gtest gflags
* csparse

### 2. build 
* git clone 
* mkdir build && cd build
* cmake .. && make -j8

### 3. prepare
* kitti_00 dataset
* orb vocabulary zip

### 4. config
* edit ./config/default.yaml (set right path)
